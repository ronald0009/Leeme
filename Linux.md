### Comandos para administrar el servicio

```
history

ps aux | grep java

sudo kill -9 13563

sudo /~/serviceName start

tail -f /~/logFile
```

### Buscar el archivo por nombre

```
find . -name FILE_NAME
```

### Listar procesos Java

```
ps aux | grep java
```

### Buscar texto

```
grep -rnw . -e 'fit.bpm.repository'
```
