### Comandos para administrar el servicio

```
history

ps aux | grep java

sudo kill -9 13563

sudo /~/serviceName start

tail -f /~/logFile
```

```
find . -name FILE_NAME
```

```
ps aux | grep java
```

```
grep -rnw . -e 'fit.bpm.repository'
```
