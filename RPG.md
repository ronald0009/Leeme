### Rutina Principal

```cobol
 *
C                     MOVE '1'       *INLR
C                     RETRN
 *
```

### Rutina Inicial

```cobol
 *
C           *ENTRY    PLIST
C                     PARM           PARM1 256
 *
C                     ENDSR
 *
```

### Inicializar una variable tipo cadena

```cobol
 *
C                     MOVE *BLANKS   CHARVAR
 *
```

### Inicializar una variable tipo numerico

```cobol
 *
C                     MOVE *ZEROS    NUMVAR
 *
```
