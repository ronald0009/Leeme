# Buscar elementos por notepad++
Agregar los siguientes sufijos al nombre:
```
{<., ')}
```
# Buscar elementos
Usar las combinaciones:
1. Abrir prompt de buscar
2. Buscar anterior
3. Buscar siguiente
```
F3
```
# Buscar etiquetas
Usar las combinaciones:
1. Abrir prompt de buscar/reemplazar
2. Buscar anterior
3. Buscar siguiente
```
ALT + F7
ALT + F8
ALT + F9
```
